# 🚀 Extensiones recomendadas para Visual Studio Code

Este repositorio contiene una lista de extensiones útiles para mejorar la productividad y la experiencia de desarrollo en Visual Studio Code.

## 📦 Extensiones

### 1. [CodeSnap](https://marketplace.visualstudio.com/items?itemName=adpyke.codesnap)
Permite generar capturas de código con un diseño atractivo para compartir en redes sociales o documentación.

### 2. [CSS Peek](https://marketplace.visualstudio.com/items?itemName=pranaygp.vscode-css-peek)
Facilita la navegación entre archivos CSS y HTML, permitiendo ver estilos aplicados directamente en el código.

### 3. [Discord Presence](https://marketplace.visualstudio.com/items?itemName=icrawl.discord-vscode)
Muestra tu actividad en Visual Studio Code dentro de Discord, incluyendo el archivo en el que estás trabajando.

### 4. [Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)
Resalta errores y advertencias directamente en el código, mejorando la detección y corrección de problemas.

### 5. [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)
Un asistente de inteligencia artificial que sugiere código en tiempo real para mejorar la velocidad de desarrollo.

### 6. [Material Design Icons](https://marketplace.visualstudio.com/items?itemName=tal7aouy.icons)
Conjunto de iconos para mejorar la apariencia del explorador de archivos en VS Code.

### 7. [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
Inicia un servidor local para visualizar cambios en tiempo real en archivos HTML y CSS.

### 8. [Language Pack - Español](https://marketplace.visualstudio.com/items?itemName=MS-CEINTL.vscode-language-pack-es)
Traduce la interfaz de Visual Studio Code al español.

### 9. [Indent Rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow)
Colorea la indentación del código para mejorar la legibilidad, especialmente en archivos con sangría profunda.

### 10. [Copilot Theme](https://marketplace.visualstudio.com/items?itemName=BenjaminBenais.copilot-theme)
Un tema visual inspirado en GitHub Copilot con colores suaves y agradables para la vista.

---

## 💡 Instalación
Puedes instalar estas extensiones directamente desde el [Marketplace de Visual Studio Code](https://marketplace.visualstudio.com/vscode) o utilizando el siguiente comando en la terminal de VS Code:

```sh
code --install-extension <nombre-de-la-extensión>
```

Ejemplo:
```sh
code --install-extension adpyke.codesnap
```
